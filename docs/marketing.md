# 마케팅 전략

## 타겟 고객

### 1차 타겟: "공유형 MZ" (핵심)

```
나이:    18~28세
성별:    여성 65% (공유 주도층)
직업:    대학생, 사회초년생
플랫폼:  카카오톡 > 인스타 > 에브리타임 > X
행동:    카톡 단톡 링크 공유 주 3-5회, 인스타 스토리 일 1회+
심리:    자기 정체성 탐색, 관계 속 포지셔닝, 대화 소재 갈증
```

```
역할:    "이거 진짜 소름이야" 2차 확산 게시물 생성
```

### 3차 타겟: "기업 마케터" (B2B, Phase 2)

```
역할:    브랜디드 테스트 의뢰 (50~200만원/건)
```

## 채널 전략

### Tier 1: 카카오톡 (바이럴 엔진) — 최우선

```
1. 공유 최적화
   - OG 이미지: 1200x630, Earthy 그라데이션 + 유형 이모지+이름 크게
   - 공유 메시지: "나는 [유형명]이래 ㅋㅋ 너도 해봐!"
   - 버튼: "나도 테스트하기"

2. 오픈채팅방 시딩
   - 대학교 학과별, 동아리, 취미, 직장인 채팅방
   - 방법: 직접 해보고 결과 캡쳐와 함께 자연스럽게 공유
   - "이거 진짜 소름이야 ㅋㅋ" 후기 형태
   - 광고성 티 나면 역효과 → 진짜 유저처럼

3. 카카오 채널 (향후)
   - "테스티피" 채널 → 새 테스트 알림 발송
```

### Tier 2: 에브리타임 (대학생 직접 공략)

```
- "OO대 학생 유형 테스트" 학교별 특화
- 시험기간 "시험기간 생존 유형" 시의성 테스트
- 결과 캡쳐 이미지 첨부 → 클릭률 3배
- 주의: 광고 글 신고 시 삭제됨 → "내가 해봤는데" 형태 필수
```

### Tier 3: 인스타그램 (시각적 바이럴)

```
- 결과 이미지 인스타 스토리 공유 유도 (@testifi.kr 태그)
- 릴스: "이 테스트 결과가 너무 정확해서 소름" 15초 리액션
- 캐러셀: "유형별 특징 한눈에 보기" 저장 유도
```

### Tier 4: X(트위터) (밈 확산)

```
- 결과를 밈 포맷으로 트윗
- "이 유형 나온 사람 RT" 참여 유도
```

### Tier 5: 네이버 블로그 (SEO)

```
- "심리테스트 추천", "재미있는 테스트" 키워드
- 네이버 서치어드바이저 사이트맵 등록
```

## 바이럴 루프

```
K = i × c  (i=초대 수, c=전환율)
목표: K > 0.5 (자연 성장), K > 1.0 (폭발 성장)

i 높이기:
  - 공유가 가장 자연스러운 동선 (결과 → 바로 공유)
  - 카카오 버튼 가장 크게
  - 유형명을 공유하고 싶게 (유머+공감)
  - 궁합: "너랑 나 궁합 봐봐!" 추가 공유 유도

c 높이기:
  - 카카오 썸네일 매력적 (Earthy 그라데이션 OG)
  - 랜딩 → 테스트 시작: 클릭 1번 (로그인/약관 없음)
  - 로딩 2초 이내
```
