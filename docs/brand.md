# 브랜드 가이드: TestiFi

## 브랜드 이름

**TestiFi** (테스티피)

- **Testi**: Test(테스트) + Testify(증명하다)의 조합. "너 자신을 증명해봐."
- **Fi**: Wi-Fi처럼 연결의 느낌. 사람과 사람을 테스트로 연결.
- **발음**: 테스티파이. 줄여서 테스티피.

### 표기 규칙

```
공식:       TestiFi
한글:       테스티피
줄임:       TF (로고/파비콘)
URL:        testifi.kr (또는 testifi.vercel.app)
해시태그:   #테스티피 #TestiFi #심리테스트
슬로건:     "너를 알아가는 가장 가벼운 방법"
```

## 브랜드 아이덴티티

### 핵심 가치

| 키워드 | 설명 |
|--------|------|
| **산뜻함** | 무겁지 않다. 가볍게 시작하고 가볍게 공유한다. |
| **자연스러움** | 억지스러운 광고 느낌 없이, 친구가 보내준 링크처럼. |
| **발견** | "아 나 이런 면이 있었어?" 작은 깨달음의 즐거움. |
| **연결** | 테스트 결과를 나누며 서로를 알아가는 과정. |

### 브랜드 퍼소나

```
이름: 티피 (TiFi)
나이: 24세
성격: 단톡방에서 재미있는 링크를 제일 먼저 보내는 친구.
     가볍게 웃기면서도 은근히 핵심을 찌르는 타입.
말투: 반말 + 친근함 + 이모지 자연스럽게
     예) "ㅋㅋ 이거 진짜 너야" / "솔직히 좀 찔리지? 🌿"
```

### 브랜드가 하지 않는 것

```
❌ 무거운 심리학 용어 (프로이트, 무의식 등)
❌ 정확한 심리 진단 약속 (의료 영역 침범)
❌ 부정적 결과 (모든 유형이 매력적이어야)
❌ 가르치는 톤 ("당신은 ~해야 합니다")
❌ 과한 색상/이펙트 (Earthy의 차분함 유지)
```

### 카피라이팅

**테스트 제목:**
```
✅ "2026 나의 연봉 유형 테스트"

❌ "심층 성격 분석 테스트"

**질문:**
```
✅ "금요일 퇴근 후 갑자기 3시간이 생겼다! 뭐 할래?"
❌ "당신은 계획적인 편입니까?"
```

**결과:**
```
✅ "조용히 통장 잔고 확인하면서 미소 짓는 타입이에요.
    티 안 내지만 친구들 중에 제일 알뜰하죠?

❌ "당신은 내향적이며 계획적인 성격입니다."
